# 🔥 Dating Profile Optimizer

Transform your dating bio into a match magnet with AI. Get viral-worthy profiles in seconds.

## ✨ Features

✅ Upload photos (AI analyzes vibe)
✅ Paste your current bio
✅ Get 5 unique viral-worthy variations
✅ Copy & share instantly
✅ 100% private (API key never stored)
✅ Works on Tinder, Hinge, Bumble, etc.

---

## 🚀 Quick Start (5 minutes)

### 1. Get Anthropic API Key (FREE)
- Go to https://console.anthropic.com/keys
- Sign up (free tier includes credits)
- Copy your API key

### 2. Clone & Setup Locally

```bash
# Clone this project
git clone <your-repo-url>
cd dating-profile-optimizer

# Install dependencies
npm install

# Run locally
npm run dev
```

Then open http://localhost:3000 and paste your API key when prompted.

### 3. Deploy to Vercel (FREE)

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

When Vercel asks about environment variables, skip it (users will input their own API key).

**That's it! You now have a live website.** 🎉

## 🛠️ How It Works

1. User uploads photos + bio
2. Claude analyzes tone/vibe
3. Generates 5 unique variations:
   - Mysterious
   - Funny
   - Ambitious
   - Chill
   - Romantic
4. Each includes why it works
5. User copies & uses on dating app
